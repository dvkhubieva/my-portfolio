# Проект "Цветочная лавка FlowerBar"

**Цель:** Проведение исследований и анализ пользовательских сценариев для улучшения продаж цветочного бутика.

## Задачи
1. **Customer Development (CustDev):**
   - Проведение интервью с клиентами.
   - Анализ степени удовлетворенности.
   - Результаты в файлах:
     - [CustDev FlowerBar (xlsx)](./CustDev_Flowerbar.xlsx)
     - [Customer Development исследования (pdf)](./Customer_Development_исследования.pdf)

2. **A/B тестирование:**
   - Гипотеза: Добавление блока "сопутствующие товары" увеличит продажи.
   - План и метрики:
     - [Практика: гипотеза и метрики эксперимента (xlsx)](./Практика_гипотеза_и_метрики_эксперимента.xlsx)

3. **Анализ продаж и Unit-экономики:**
   - Оценка бюджета и точек роста.
   - Подробности:
     - [Работа с бюджетами (xlsx)](./Работа_с_бюджетами_FlowerBar.xlsx)
